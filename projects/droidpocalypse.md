Droidpocalypse is a robot survival game created for a high school capstone project using Godot v3.5.1. The game is heavily inspired by the Roguelike Indie game Vampire Survivors created by poncle, and thus has similar gameplay. 

I programmed the game using Godot's GDScript (based on Python), and had a lot of fun solving all the unique challenges it presented, from figuring out how to create all the unique weapons to the creative challenge of coming up with unique enemy characters.

This project gave me a newfound appreciation for the technical and creative feat that creating a game is, and I've only enjoyed other games more since.
